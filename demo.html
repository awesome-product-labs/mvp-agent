Ready to Get Started?

The MVP Generation Agent is ready to help you build better products faster.

Quick Setup:

# 1. Clone or navigate to the project
cd mvp-generation-agent/backend
# 2. Activate virtual environment
source venv/bin/activate
# 3. Test the system
python test_mvp_agent.py
# 4. Start the API server (with Claude API key)
uvicorn src.api.main:app --reload

Note: To use real AI analysis, get a Claude API key from Anthropic and update the ANTHROPIC_API_KEY in your .env file.
